# Moscow Time Web App

## Description
Web app which show current Moscow time.\
DevOps course, Innopolis University

## Start
### Pre-install
- `Python 3.9`
- `flask` library - framework
- `pytz` library - for getting Moscow time

### Run
`python app.py`
